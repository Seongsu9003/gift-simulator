## 1. 제품 개요

### 1.1 배경 및 목적

삼성자산운용 KODEX ETF 증여 가이드북에 따르면, 자녀에게 조기 증여를 시작할수록 복리 효과와 증여세 절세 혜택이 극대화됩니다. 특히 미성년 자녀에게는 10년 단위로 2,000만 원(만 19세 미만)의 증여세 면제 한도가 적용되며, 조기에 시작할수록 면세 혜택을 더 많이 받을 수 있습니다.

대부분의 부모들은 증여 계획 수립 방법, 절세 전략, ETF 상품 선택, 행정 절차 등 복잡한 요소들로 인해 증여를 미루거나 잘못된 방식으로 진행합니다.

본 서비스는 자녀의 출생부터 성인이 되는 30년간의 ETF 증여 계획을 쉽고 직관적으로 수립·관리할 수 있는 웹 플랫폼을 제공합니다.

### 1.2 서비스 비전

> **"내 아이의 금융 미래를 30년 플랜으로 설계하다"**

증여 계획 수립부터 절세 시뮬레이션, 수익률 예측까지 — 부모가 자녀의 경제적 미래를 위해 필요한 핵심 기능을 빠르게 검증합니다.

### 1.3 핵심 가치 제안 (Value Proposition)

- 간편한 30년 증여 로드맵 자동 생성
- 증여세 절세 효과 실시간 시뮬레이션
- 월 적립식 복리 수익 예측 (연 10.23% 고정 기준)

### 3.2 Firebase Google 로그인 및 복수 자녀 지원 (v1.1.0 추가)

**기능 설명**

사용자는 Google 계정으로 로그인하여 여러 자녀 정보를 등록하고 관리할 수 있습니다. 각 자녀별로 개별적인 증여 플랜을 생성하고 전환하여 확인할 수 있습니다.

**핵심 기능**

| 기능 | 설명 | 구현 상태 |
|---|---|---|
| Google OAuth | Firebase Auth 기반 Google 로그인 | ✅ |
| 자녀 정보 관리 | 이름, 생년월일, 성별, 관계 등록/수정 | ✅ |
| 복수 자녀 지원 | 여러 자녀 등록 및 카드형 목록 표시 | ✅ |
| 자녀 전환 | 헤더에서 자녀 선택 버튼으로 빠른 전환 | ✅ |
| 데이터 영구 저장 | Firestore users/{userId}/children/{childId} | ✅ |

**Firestore 데이터 구조**

```
users/
  {userId}/
    children/
      {childId}/
        - name: "홍길동"
        - birthDate: "2020-03-15"
        - gender: "male" | "female"
        - relationship: "child" | "grandchild"
        - createdAt: timestamp
        - updatedAt: timestamp
```

**UI/UX 특징**

- 자녀 카드에 성별 이모지 표시 (👦👧)
- 자녀 전환 버튼: "홍길동 👦 ▼"
- 자녀가 없으면 자동으로 자녀 추가 폼 표시
- 기존 자녀 데이터 자동 마이그레이션 지원

### 3.3 30년 증여 로드맵 생성기

**기능 설명**

자녀의 생년월일, 현재까지 증여한 금액, 월 적립 가능 금액을 입력하면 법적 증여세 면제 한도에 맞춰 30년간의 최적 증여 계획을 자동으로 생성합니다.

**입력값**

| 입력 항목 | 타입 | 필수 여부 | 설명 |
|---|---|---|---|
| **자녀 이름** | Text | 필수 | 자녀 식별용 (v1.1.0 추가) |
| 자녀 생년월일 | Date | 필수 | 만 나이 자동 계산 |
| 자녀와의 관계 | Select | 필수 | 직계비속 (자녀/손자녀) |
| **자녀 성별** | Select | 필수 | 남자아이/여자아이 (v1.1.0 추가) |
| 현재까지 증여 금액 | Number | 선택 | 기존 증여 이력 반영 |
| 월 적립 희망 금액 | Number | 필수 | 자동이체 기준 금액 |

**출력값 — 로드맵 타임라인**

- 만 0~10세: 미성년 1구간 (면제 한도 2,000만 원)
- 만 10~19세: 미성년 2구간 (면제 한도 2,000만 원)
- 만 19~29세: 성인 구간 (면제 한도 5,000만 원)

**화면 구성**

- 타임라인 인포그래픽: 10년 단위 구간별 증여 금액 시각화
- 누적 증여 금액 vs 면제 한도 진행 바
- 구간별 예상 세금 절감액 표시

### 3.3 증여세 시뮬레이터

**기능 설명**

증여 금액, 관계, 기간에 따른 세금을 자동 계산하고, 면제 한도 초과 시 세율 구간과 납부 세액을 명확하게 보여줍니다.

**증여세 면제 한도 기준 (10년 단위)**

| 수증자 관계 | 10년 면제 한도 | 초과 시 세율 |
|---|---|---|
| 직계비속 (미성년, 만 19세 미만) | 2,000만 원 | 10% ~ 50% |
| 직계비속 (성인) | 5,000만 원 | 10% ~ 50% |
| 배우자 | 6억 원 | 10% ~ 50% |
| 기타 친족 | 1,000만 원 | 10% ~ 50% |

**출력 UI**

- 원형 게이지: 면제 한도 사용률 (%)
- 절세 효과 비교표: 지금 증여 vs 10년 후 증여
- 세금 납부 예상 시나리오 3가지 (최소 / 중간 / 최대)

---

### 3.4 수익률 시뮬레이터

**기능 설명**

월 적립식 투자 시 연 10.23% 고정 수익률 기준으로 30년 후 예상 자산을 시뮬레이션합니다. (ETF 종목 구분 없이 단일 수익률 적용. VOO/QQQM 개별 비교는 V2에서 제공)

**시뮬레이션 파라미터**

- 월 적립 금액: 슬라이더 (1만 원 ~ 100만 원)
- 투자 기간: 5년 / 10년 / 20년 / 30년
- 연 수익률: **10.23% 고정** (역사적 평균 기반)

**출력 차트**

- 선형 그래프: 납입 원금 vs 예상 평가액 성장 곡선
- 핵심 지표 카드: 총 납입액 / 예상 평가액 / 수익 배수

## 9. 가정 및 의존성

### 9.1 핵심 가정

- 현행 증여세법 (2025년 기준) 면제 한도가 유지된다고 가정
- 수익률 시뮬레이터는 연 10.23% 고정값을 사용 (역사적 S&P 500 평균 기반)
- MVP는 비로그인 사용자 대상으로 제공 (로그인 없이 시뮬레이션 가능)
- MVP는 한국 내 서비스만 제공

### 9.2 외부 의존성

- ETF 실시간 데이터 연동: MVP에서는 고정 수익률 사용으로 외부 API 의존 없음
- 홈택스 API: 자동 연동 불가 시 화면 가이드로 대체

### 10.2 용어 정의

| 용어 | 정의 |
|---|---|
| 증여 | 타인에게 재산을 무상으로 이전하는 법률 행위 |
| 증여세 면제 한도 | 10년 단위로 세금 없이 증여 가능한 최대 금액 (미성년 자녀: 2,000만 원, 성인 자녀: 5,000만 원) |
| ETF (상장지수펀드) | 주식시장에 상장되어 거래되는 인덱스 펀드 |
| 복리 효과 | 원금에 이자가 더해지고, 그 합계에 다시 이자가 붙는 효과 |
| 적립식 투자 | 일정 금액을 정기적으로 분할하여 투자하는 방식 |
| VOO | Vanguard S&P 500 ETF (V2에서 개별 비교 지원 예정) |
| QQQM | Invesco Nasdaq-100 ETF (V2에서 개별 비교 지원 예정) |
| S&P 500 | 미국 대표 500개 기업으로 구성된 주가지수 |
| Nasdaq-100 | 미국 나스닥 시장 상위 100개 기업으로 구성된 주가지수 |
